# Student Performance Analysis & Prediction

This project looks at student scores and predicts if a student will **pass or fail**.

## What the Project Does
- Loads a small dataset of students
- Shows basic information and statistics
- Creates graphs to visualize scores, attendance, and study hours
- Builds a simple machine learning model to predict pass/fail
- Can predict for a new student

## Files in the Project
- `students.csv` – the dataset  
-  `part4_visualization_ml.ipynb` – the Python code/notebook  
- `plot1_bar.png` … `plot7_seaborn_scatter.png` – the graphs generated by the code  

## How to Run
1. Open the Python file or Jupyter Notebook
2. Make sure `students.csv` is in the same folder
3. Run the code step by step
4. See the results and graphs
